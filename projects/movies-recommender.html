Billiards

Goal

Create a fun game where you can play the game Billiards with realistic collisions and physics.

Process

Used C# and .NET windows forms to create a GUI window. Produced a user-friendly user interface with intuitive controls

Outcome

A Simple Billiards game that you can play with your friends on the same device. I learned about GUIs, and about how physics engines work, implementing features such as collision detection and resolution.

Technologies Used

C#
.NET
Windows Forms
